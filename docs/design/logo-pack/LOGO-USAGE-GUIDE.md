# Runa Hafıza Mark Paket Rehberi

Bu paket `Hafıza` markasının üretim için hazırlanmış 6 SVG varyantını içerir:

- `runa-hafiza-mark-micro-brand.svg`
- `runa-hafiza-mark-regular-brand.svg`
- `runa-hafiza-mark-bold-brand.svg`
- `runa-hafiza-mark-micro-mono.svg`
- `runa-hafiza-mark-regular-mono.svg`
- `runa-hafiza-mark-bold-mono.svg`

## Kullanım Eşlemesi

- `micro`:
  - 12x12 (mobil mesaj başı)
  - 16x16 (çok sıkışık toolbar veya chip)
- `regular`:
  - 18x18 (masaüstü mesaj başı)
  - 24x24 (sidebar küçük alanlar)
  - 32x32 (ikincil hero)
- `bold`:
  - 52x52 (auth/empty state hero)
  - 60x60 (onboarding odak görseli)
  - 64x64 (yüksek vurgu alanları)

## Variant Kuralı

- `brand`: ürün içi varsayılan.
- `mono`: tek renk baskı, düşük kontrast yüzeyler, ikon butonları ve durum bağımsız arayüz elemanları.
