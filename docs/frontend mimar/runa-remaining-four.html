Bölüm 01

Görsel oluşturma akışı

Sohbet içinden tetiklenir — ayrı mod veya sekme yok. Kullanıcı "bu fotoğrafı düzenle" veya "bir afiş oluştur" der; Runa tool satırıyla başlar, sonuç doğrudan sohbet akışına gömülür. Varyasyon · Düzenle · Yeniden Oluştur · Kaydet aksiyonları sonucun altında durur — dashboard değil, devam eden sohbet.

Tasarım kararları
Görsel üretme bir tool aktivitesi — "görsel oluşturuluyor ›" tek satır, açılınca prompt + ayarlar görünür. Asistan mesajından önce gelir.
Üretilen görsel compact hâlde (180–220px kare) sohbete gömülür. Tıklanınca tam boyuta açılır — ayrı ekran yok, sheet yok.
Yeniden oluşturma: kullanıcı doğal dille yazar ("daha koyu yap", "siluet kalsın"). Teknik prompt editörü yoktur.
3 varyasyon küçük kare olarak altında durur; seçilenin border'ı accent rengine döner.
Kaydet → Runa dosyaya yazar. Yüksek-risk (üzerine yazma) → inline approval devreye girer.

Bölüm 03

Çoklu dosya yönetimi

Dosya işlemleri sohbet içinde kalır — ayrı panel yok. Tool satırı "3 dosya okundu ›" açılınca liste + seçme + arama + toplu işlem. Dosya değişikliği sonrası diff görünümü inline açılır. Büyük topluluk işlemleri yüksek-risk onay gerektirir.

Dosya işlem kalıpları
Düşük risk Dosya okuma, listeleme, arama → onaysız, tool satırında özetlenir
Orta risk Tek dosya yazma/değiştirme → inline onay, hedef dosya gösterilir
Yüksek risk Toplu silme, dizin temizleme → uyarı bandlı inline onay, geri alınamaz notu
Diff görünümü her zaman collapsible → varsayılan kapalı, kullanıcı isterse açar

Bölüm 04

Bildirimler & arka plan görevleri

İki kategori: Hatırlatmalar (kullanıcının söylediği, Runa'nın kaydettiği) ve Sistem (görev bitti, onay bekliyor, cihaz hatası). Bildirim merkezi mobilde sohbet header'ından açılan bir sheet, masaüstünde sağ kenardan açılan bir panel. Arka planda çalışan görevler her zaman görünür; sohbeti kilitlemez.

Hatırlatma akışı
Kullanıcı sohbette "yarın saat 10'da toplantımı hatırlat" der → Runa onay ister ("kaydedeyim mi?") → kullanıcı onaylar → kaydolur → zamanı gelince OS push bildirimi, tıklayınca doğrudan ilgili sohbete düşer.
Hatırlatmalar silinebilir, ertelenebilir (10dk / 1s / yarın)
Sistem bildirimleri ayarlardan kategori bazında kapatılabilir
Sessiz saatler tüm kategorilere uygulanır
